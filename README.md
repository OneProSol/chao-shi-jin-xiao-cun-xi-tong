### 一、相关技术
- 前端：`thymeleaf`、`HTML` / `CSS` / `Javascript`。
- 后端：`Springboot`等。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
### 三、部署所需环境

- 后端：`jdk1.8`。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示
系统分为：管理员。

管理员功能：1. 登录、2. 客户管理、3. 供应商管理、4. 商品管理、5. 商品类别管理、6. 商品进货、7. 商品退货查询、8. 商品销售、9. 商品销售退货、5. 权限管理、6. 角色管理、7. 用户管理、8. 日志管理、9. 统计报表
